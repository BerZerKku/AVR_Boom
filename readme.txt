Бомба для квестов.

Написана программа для AT90S2313. Железо сделано на отладочной плате.

Подключение:
PORTD0-PORTD3 - символы индикатора
PORTB0-PORTB6, PORTD4 - разряды индикатора в порядке от а,b,c,d,e,f,g,point

Так же через два резистора по 10кОм соединины пары выводов:
PORTB4 ->10кОм - перемычка - 10кОм -> PORTB0
PORTB5 ->10кОм - перемычка - 10кОм -> PORTB1
PORTB6 ->10кОм - перемычка - 10кОм -> PORTB2
PORTD4 ->10кОм - перемычка - 10кОм -> PORTB3

Проверка проводится каждый тик смены символа, т.е. каждые 2 мс. В это время напряжение с символов снимается.
